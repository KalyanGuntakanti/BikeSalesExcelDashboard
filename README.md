# 🚲 Bike Sales Dashboard

An interactive Excel dashboard that analyzes bike purchase behavior across customer demographics, income levels, and commute patterns.

---

## 📊 Project Overview

This project explores what drives a customer to purchase a bike. Using a dataset of 1,000+ customer records, it uncovers purchase trends by income, age, commute distance, education, occupation, and region — all visualized in a clean, interactive Excel dashboard.

---

## 📁 File Structure

| Sheet | Description |
|---|---|
| `bike_buyers` | Raw dataset with 13 customer attributes |
| `Working Sheet` | Cleaned and transformed data used for analysis |
| `Pivot Table` | Aggregated summaries powering the dashboard |
| `Dashboard` | Visual summary with charts and slicers |

---

## 🗂️ Dataset Columns

| Column | Description |
|---|---|
| `ID` | Unique customer identifier |
| `Marital Status` | Married (M) or Single (S) |
| `Gender` | Male (M) or Female (F) |
| `Income` | Annual income in USD |
| `Children` | Number of children |
| `Education` | Highest education level |
| `Occupation` | Job category (Professional, Clerical, Manual, etc.) |
| `Home Owner` | Whether the customer owns a home |
| `Cars` | Number of cars owned |
| `Commute Distance` | Daily commute range (0–1 Miles to 10+ Miles) |
| `Region` | Europe or Pacific |
| `Age` | Customer age |
| `Purchased Bike` | Target variable — Yes or No |

---

## 🔍 Key Insights

- **Income** is a strong predictor — higher-income customers show different purchase patterns
- **Commute distance** influences purchases, with short-commute customers more likely to buy
- **Age brackets** reveal distinct buying behavior across young adults, middle age, and seniors
- **Region** shows notable differences between European and Pacific customer bases

---

## 🛠️ Tools Used

- **Microsoft Excel** — Data cleaning, pivot tables, slicers, and dashboard design
- **Excel Charts** — Bar, line, and clustered column charts
- **Pivot Tables** — Dynamic aggregations by demographic segments

---

## 🚀 How to Use

1. Download `Bike_Sales_Dashboard.xlsx`
2. Open in Microsoft Excel (2016 or later recommended)
3. Navigate to the **Dashboard** tab
4. Use the slicers to filter by region, marital status, or education level
5. Charts update automatically based on your selections
